CSS

E para dar alguma vida à página, ela usa o CSS3!

Seletor Utilização (👍/👎) Funcionamento/Utilização
elemento 👍 Seleciona os elementos que são do tipo especificado
#id 👍 Seleciona o elemento com o id respetivo
.classe 👍 Seleciona os elementos que têm a classe
[atributo] 👍 Seleciona os elementos que possuem o atributo especificado
Combinadores de seletores Utilização (👍/👎) Funcionamento/Utilização
Lista (,) 👍 Aplica a regra a todos os seletores especificados
Descendente ( ) 👎 Aplica a regra aos descendentes selecionados do elemento especificado
Filho (>) 👍 Aplica a regra aos filhos imediatos selecionados do elemento especificado
Adjacentes (~) 👎 Aplica a regra a elementos que partilhem o mesmo elemento pai
Imediatamente Adjacentes (+) 👍 Aplica a regra a elementos que estejam adjacentes
